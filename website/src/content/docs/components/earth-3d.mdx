## Usage
### Basic Earth model
Basic render of the Earth3D model. The radius and the base map or the render texture can be altered.
You can also provide your own base, specular and normal maps to the model.
<Earth3DExample/>

```javascript
import React from "react";
import { Earth3D, Earth3DGSON } from "@zephyr3D/react";
import { Canvas } from "@react-three/fiber";
import { OrbitControls } from "@react-three/drei";


export const Earth3DExample = () => {
  return (
    <div className="bg-white/5 h-72">
      <Canvas>
        <ambientLight />
        <directionalLight intensity={1} position={[2, -2, 2]} />
        <Earth3D textureType={"base"} />
        <OrbitControls />
      </Canvas>
    </div>
  );
};
```

### Using GeoJSON
GeoJSON data can be rendered on the Earth3D model. 

<Earth3DGSONExample/>

```javascript
import React from "react";
import { Earth3D, Earth3DGSON } from "@zephyr3D/react";
import { Canvas } from "@react-three/fiber";
import { OrbitControls } from "@react-three/drei";
import { EffectComposer, Bloom } from "@react-three/postprocessing";

export const Earth3DGSONExample = () => {
  return (
    <div className="bg-white/5 h-72">
      <Canvas>
        <ambientLight />
        <directionalLight position={[2, -2, 2]} />
        <Earth3DGSON gsonPath="/countries.geo.json"/>
        <OrbitControls />
        <EffectComposer>
          <Bloom
            luminanceThreshold={0.3}
            luminanceSmoothing={0.9}
            intensity={1}
          />
        </EffectComposer>
      </Canvas>
    </div>
  );
};
```
